# SUA-Bot - Roleplay Organization

WhatsApp Bot specialized in roleplay team organization.

## Features
- Character Assignment
- Activity Tracking
- Anti-Spam System
- Warning System
- Suggestions & Excuses

## Deployment on Hugging Face
This space runs on Docker. Make sure to set the `MONGO_URI` secret in the Space settings.
